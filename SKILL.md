# AIGC降率改写技能(学术版)

将课题申报书/论文的AIGC率从原始80%+稳定降至25-35%区间,适用维普2026年多模型交叉验证检测。

## 输入获取

如果用户没有提供具体文本,询问以下信息:
1. 待降率文本(直接粘贴或文件路径)
2. **文本所属章节类型**: 摘要 / 引言绪论 / 文献综述 / 研究设计 / 结果分析 / 结论 / 致谢 / 局限 - 章节类型决定改写幅度
3. 目标AIGC率(默认 25-35%)
4. 目标检测平台(维普/知网/朱雀,默认维普;若多平台都要过,优先按朱雀标准改)
5. 是否有可用的本地数据/访谈素材/统计结果(用于人工注血提示)

## 一、维普核心原理

### 1.1 五大指纹战场(基础维度)

不查"抄没抄",而是文本的**统计特征指纹**:
- 句法结构(30%): 长短句交替/语态混用/插入不完整句
- 段落连贯性(25%): 制造思维跳跃,避免0.7-0.8的丝滑过渡
- 信息密度(20%): 段落胖瘦不均
- 词汇多样性(15%): TTR动态波动
- 标点与主体性(10%): 第一人称+情绪+口语

### 1.2 章节加权机制(关键!)

维普对不同章节赋予不同权重,**改写幅度必须按权重分配**:

| 章节类型 | 权重系数 | 改写幅度 | 说明 |
|---------|---------|---------|------|
| **摘要(中英文)** | ×1.8 | 80% (深度重构) | 影响最大,优先改 |
| **引言/绪论** | ×1.5 | 80% (深度重构) | 与结论呼应改 |
| **研究结论** | ×1.5 | 80% (深度重构) | 与引言呼应改 |
| **创新点阐述** | ×1.5 | 80% (深度重构) | |
| 文献综述 | ×1.0 | 60% (批判性注入) | 不改陈述,加批判 |
| 研究设计/方法 | ×1.0 | 60% (标准改写) | |
| 结果分析 | ×1.0 | 60% (加数据解释) | |
| 研究背景 | ×0.5-0.8 | 30% (轻度) | 主要保持风格统一 |
| 理论基础 | ×0.5-0.8 | 30% (轻度) | |
| 研究局限 | ×0.5-0.8 | 30% (轻度) | |
| 致谢 | ×0.5-0.8 | 30% (轻度) | |

### 1.3 双维度评分

维普用两个维度:
- **维度A**: 整体AIGC疑似度(0-100%) — 各章节AIGC率×权重的加权平均
- **维度B**: 关键章节风险等级(低/中/高/极高) — 即使整体达标,关键章节"极高"仍可能被退回

**实操含义**: 不能只盯整体率,必须确保摘要/引言/结论各自单独达标。

### 1.4 拼接预警机制(独特,易踩坑!)

跨章节风格不统一会触发"拼接预警",**全文风格必须保持基本一致**:
- ❌ 章节间句长差异过大(如引言平均18字 vs 文献综述28字)
- ❌ 人称切换(第三人称突然转第一人称)
- ❌ 用词复杂度突变(简洁直白突然变复杂从句)

**实操含义**: 改写时不能某些章节深度改、某些章节不改;低权重章节也要轻度改保持风格。

### 1.5 数据密集型容忍

维普对**含具体数据的段落容忍度高**(AIGC率仅15-25%):
- β系数、p值、R²、t值、F值、置信区间等**主动写出来**
- 具体百分比、绝对数字、比率
- 表格、图注、统计描述

**实操含义**: 不只是"模糊数据",凡能写精确数据的地方就写精确数据,反而降率。

### 1.6 引用规范性

无引用的学术陈述会被+10-15%:
- 综述部分需保留[1][2]或(Smith, 2020)等规范引用
- 介绍既有研究时必须有引用支撑
- 自己的判断可以不用引用(反而提升人类感)

## 二、操作流程: AI洗稿 → 标注人工补充 → 分析指标 → 章节交叉检查 → 输出

### 第一步: AI洗稿(由你执行)

按以下**六大维度**进行**破坏性改写**(章节1-5基础,章节6学术专用):

#### 1) 句法破坏
- 所有>50字的句子拆为2-3句
- 每段保证至少1个短句(≤15字)和1个长句(≥40字)
- 语态分布: 主动60% / 被动20% / 倒装或省略主语 20%
- 插入1-2个"非完整句": "值得注意的是..."、"一个常被忽略的问题是..."
- **章节间句长一致**: 各章节平均句长差控制在 ±5字内,避免拼接预警

#### 2) 结构破坏
- 砍掉"一是/二是/三是"、"首先/其次/再次"等并列结构(机械列举AIGC率可达85%+)
- 在"总-分-总"中插入至少1处思维跳跃:
  - 设问: "这一点是否成立?"
  - 回溯: "回到前文提到的..."
  - 岔开: "与此相关但常被忽略的是..."
  - 反驳: "这个判断可能过于乐观,但..."
- 段落胖瘦不均: 高密度段3-4论点压缩,低密度段只展开1个细节
- 并列项硬上限: 不超过3项,超过则砍或拆段
- 数字不工整: 不要恰好4个目标、5个机制、6个场景

#### 3) 主体性注入
- 第一人称密度: 每300字至少1次"课题组/我们/笔者"
- **人称统一**: 全文统一使用一种人称("课题组"或"我们",不混用),避免拼接预警
- 每段至少1个判断句(含"认为/判断/指出/值得注意的是")
- 每500字至少1个情绪/评价词("令人担忧/值得欣慰/坦率地说")
- 标点比例: 逗号:句号 控制在 2:1 ~ 2.5:1 (AI常达3-4:1)
- 增加破折号(——)、括号、分号的使用

#### 4) 词汇替换
- 砍AI爱用词: 显著/重要/关键/进一步/系统/机制/全面/深入/持续/切实/充分
- 替换为模糊限定: "某种程度上"、"大致上"、"粗略判断"、"至少在...语境下"
- 至少植入1个俗语: "扎下根来"、"纸面规划"、"无米之炊"、"第一个吃螃蟹"、"话说回来"
- 四字对仗打散: "一核牵引、两翼支撑、多点示范" → 自然语言解释,保留概念但破坏形式美
- 同一概念用不同术语轮换: 机制/架构/范式/框架/模式
- **植入学科术语**(单次但精准): 社会学(场域/惯习/布迪厄)、经济学(帕累托改进/纳什均衡)、教育学(脚手架/最近发展区)等

#### 5) 数据密集化(维普特别有效)
- 在结果分析、结论部分**主动写精确数据**: β=0.47, p<0.01, R²=0.62, n=384, F(2,381)=12.4
- 用具体数据替代笼统描述: "效果显著" → "干预组比对照组高出 23.4%"
- 添加"超出预期"的数据评价: "结果令人意外——系数0.47显著高于预期的0.3"
- 引用规范化: 综述部分保留 [1] [2] 或 (Smith, 2020) 引用标记

#### 6) 学术批判性注入(文献综述专用)
**核心原则: 不改陈述内容,只加批判性评价**。这对维普特别有效,可使文献综述AIGC率从65%降至35%左右。

在每段引述既有研究后,加入:
- **样本局限批判**: "(但样本仅限制造业,能否推广到服务业不明)"
- **方法缺陷批判**: "该研究存在内生性问题,因果推断不稳健"
- **时效性批判**: "研究数据截至2018年,在数字化加速背景下的适用性存疑"
- **理论假设批判**: "假定个体决策完全理性,忽略了行为偏差"
- **结论强度批判**: "结论虽具启发性,但未在跨文化样本中复现"

## 三、降率策略组合(按场景选用)

### 策略A: 摘要优先改写法(最高ROI)
摘要权重×1.8,改它一段比改一整章管用。具体动作:
1. 打破"本研究以XX为对象,采用XX方法,得出XX结论"的标准四段式
2. 改用问题导向: 领域问题 → 现有研究不足 → 研究发现(含具体数据) → 挑战传统观点
3. 增加 1-2 个关键数据(β=0.47, p<0.01)
4. 植入1-2个学科特定术语
5. 结尾加"超出预期"或"研究局限"

### 策略B: 引言-结论呼应式改写
1. 引言明确列出 3-4 个研究问题或假设
2. 结论按引言顺序逐一回应
3. 结尾增加 1-2 句"超出预期"或"研究局限"
4. 引言和结论使用相同的关键术语,确保呼应

### 策略C: 文献综述批判性注入
**不改陈述内容**,只加方法/样本/时效批判(见 第一步 #6)。文献综述AIGC率可从65%→35%。

### 策略D: 差异化改写(全文)
按章节权重分级处理:
- 高权重(摘要/引言/结论): 深度重构,改写幅度80%
- 中权重(综述/方法/结果): 标准改写+批判注入,幅度60%
- 低权重(背景/局限/致谢): 轻度改写,幅度30%(主要拉齐风格)

### 策略E: 多平台联合策略
若需要同时通过多平台:
- **检测严格度**: 朱雀论文版 > 知网AIGC > 维普AIGC
- 用最严平台标准改写,通过则其他基本通过
- 实测案例: 朱雀专版改后,朱雀19%/知网23%/维普16%,三平台全过

### 策略F: 模型窗口期等待
维普模型每 2-3 个月更新,同篇文章不同时间检测可差 5-10%:
- 卡红线边缘(如 41%, 要求<40%): 等 1-2 周后重测可能直接通过
- 不要慌着改,先看看时间窗口

## 四、严格约束

- **绝不替用户写人工注血内容**: 必须真人手写,保持人机分离
- **章节加权差异化处理**: 摘要×1.8/引言×1.5/结论×1.5,改写幅度必须按权重分配
- **关键章节单独达标**: 整体率不是唯一指标,关键章节风险等级必须为"低"
- **拼接预警警觉**: 全文人称统一/句长一致/用词复杂度均匀,低权重章节也要轻度改
- **分阶段降率**: 先AI洗到50%,人工注血到35%,通读保留不完美到30%。不要试图一次到位
- **保留学术庄重感**: 口语化是手段,不能丢失专业性
- **数据密集化**: 凡能写精确数据(β/p/R²/n)处必写,反而降率
- **引用必须规范**: 综述部分保留 [1] (Smith, 2020) 等引用,别全砍
- **二次检测反升警告**: 不要用 ChatGPT/文心一言/Claude 改写AI内容,会强化AI特征。重构表达逻辑,不是同义词替换
- **冷处理**: 如某段反复改不下来,提示用户先放置一周,人工重写开头50字(风格锚点),再让AI续写
