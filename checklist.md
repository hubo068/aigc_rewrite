# 人工注血核验清单

> 在AI洗稿后、提交检测前,人工逐项勾选。每篇必跑。

## 章节加权层(2026新增,最关键)

不同章节权重不同,核验要求随之分级:

### 高权重章节(摘要×1.8、引言×1.5、结论×1.5)

- [ ] 摘要是否打破"本研究以XX为对象,采用XX方法,得出XX结论"的标准四段式?
- [ ] 摘要是否含 ≥1 个关键统计量(β/p/R²/n)?
- [ ] 摘要是否植入 ≥1 个学科特定术语(场域/帕累托/最近发展区等)?
- [ ] 摘要结尾是否有"超出预期"或"研究局限"?
- [ ] 引言是否明确列出 3-4 个研究问题/假设?
- [ ] 结论是否按引言顺序逐一回应?
- [ ] 结论是否含 ≥2 处个人见解("结果令人意外"、"系数X显著高于预期Y")?
- [ ] 这些章节人工注血是否加倍(数据≥6处、痕迹≥4处)?

### 中权重章节(综述/方法/结果 ×1.0)

- [ ] 文献综述每段引述后是否加批判性评价(样本/方法/时效/假设/复现)?
- [ ] 文献综述是否保留 ≥3 处规范引用([N] 或 (作者,年份))?
- [ ] 方法部分是否含具体统计指标和样本描述?
- [ ] 结果部分是否用具体数据替代笼统描述?

### 低权重章节(背景/局限/致谢 ×0.5-0.8)

- [ ] 是否做轻度改写(避免与高权重章节风格脱节)?
- [ ] 句长是否与全文保持 ±5 字内的一致性?

## 拼接预警层(2026新增)

跨章节风格统一性检查:

- [ ] 全文人称是否统一? ("我们" 或 "课题组",择一,不混用)
- [ ] 各章节平均句长差异是否 ≤5 字? (运行 cohesion 命令检查)
- [ ] 用词复杂度是否均匀? (避免某章节简洁直白、某章节复杂从句)
- [ ] 段落开头风格是否一致? (避免有的章节用"首先"、有的章节没编号)

## 引用规范层(2026新增)

- [ ] 文献综述部分引用密度 ≥3 处/千字?
- [ ] 引述既有研究是否都有 [N] 或 (作者, 年份) 支撑?
- [ ] 自己的批判/判断是否不强加引用(反而提升人类感)?

## 数据密集层(2026新增,反直觉但有效)

维普对含数据段落容忍度高(15-25%),应主动加数据:

- [ ] 结果分析部分是否含 ≥3 处精确统计(β/p/R²/t/F/n/CI)?
- [ ] 是否用具体百分比替代"显著/重要/明显"?
- [ ] 是否有 ≥1 处"超出预期"的数据评价?
- [ ] 表格/图注/统计描述是否充分?

## 数据层(降率最有效) ★★★★★

- [ ] 文中是否有 ≥3 个不精确的具体数据? ("约X家"、"X左右"、"粗略估计")
- [ ] 是否有 ≥1 个本地企业/项目/园区的具体名称?
- [ ] 是否有 ≥1 个时间锚点(年月、季度、政策节点)?
- [ ] 是否有 ≥1 处匿名访谈/调研转述?

## 痕迹层(人类指纹) ★★★★

- [ ] 是否有 ≥2 处第一人称("课题组认为/我们发现/笔者注意到")?
- [ ] 是否有 ≥1 处对困难的抱怨或质疑?
- [ ] 是否有 ≥1 处自我限定("不追求"、"受限于"、"仅就...而言")?
- [ ] 是否有 ≥1 处自我修正("一度倾向于...但...")?

## 语言层(打破AI规律) ★★★

- [ ] 是否有 ≥1 个俗语/口语词? (扎下根来/纸面规划/无米之炊/话说回来)
- [ ] 是否有 ≥1 个情绪/评价词? (令人担忧/值得欣慰/坦率地说)
- [ ] 是否有 ≥1 处括号内的离题补充?
- [ ] 是否有 ≥1 个破折号(——)制造的中断?
- [ ] 并列项是否全部 ≤3 项?
- [ ] 模糊限定词("某种程度上/大致上/粗略")是否 ≥2 处?

## 结构层(制造粗糙感) ★★★

- [ ] 是否存在 1 个"只讲1个例子"的低密度段落?
- [ ] 是否有 1 处段落开头不是承接上文?
- [ ] 是否有 1 处对前文观点的修正或反驳?
- [ ] 是否有 1 处思维跳跃("话说回来"、"与此相关但常被忽略")?
- [ ] 是否完全砍掉了"一是/二是/三是"等机械列举? (维普识别为AI模式,可达85%+)

## 句法层

- [ ] 是否有连续3句以上句长相同的情况? (如有,需打散)
- [ ] 是否全是主动语态? (如是,加入1-2句被动或倒装)
- [ ] 平均句长是否在 25-35 字之间?
- [ ] 是否有 ≥1 个 "不像学术写作" 的短句(≤15字)?

## 标点层

- [ ] 逗号:句号 比例是否在 2:1 ~ 2.5:1? (运行 analyze.py 查看)
- [ ] 是否有 ≥1 处分号(;)使用?
- [ ] 是否有 ≥1 处破折号(——)使用?

## 多平台检测策略(选用)

若需同时通过多平台,按严格度排序处理:

- [ ] 已确定主要检测平台(维普/知网/朱雀)?
- [ ] 若需多平台过,是否按朱雀(最严)标准改写?
- [ ] 改写后是否在朱雀上预检测一次?

---

## 模型窗口期策略(选用)

- [ ] 若卡红线边缘(40-45%),是否考虑等 1-2 周后重测?
- [ ] 是否记录此次检测时间,以便对比模型版本变化?

---

## 二次检测警告

- [ ] **未使用 ChatGPT/文心一言/Claude 改写AI内容**(会强化AI特征)?
- [ ] 改写采用的是**重构表达逻辑**而非同义词替换?

---

## 自查口诀

> **数据要糙,痕迹要真,语言要俗,结构要乱,标点要呼吸**
> **章节按权重,全文要一致,引用要规范,精数要敢加**

如果以上各层中有 50% 以上未勾选,直接退回重写,不要送检。
**特别是高权重章节(摘要/引言/结论)若任何一项未达标,就先别送检。**
